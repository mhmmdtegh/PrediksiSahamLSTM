Prediksi Harga Saham Menggunakan LSTM

Aplikasi berbasis Streamlit untuk memprediksi harga saham menggunakan algoritma Long Short-Term Memory (LSTM) dengan data historis dari Yahoo Finance.

Fitur

* Input kode saham
* Data historis otomatis dari Yahoo Finance
* Grafik harga historis
* Grafik harga asli vs prediksi
* Prediksi 30 hari ke depan
* Tabel hasil prediksi
* Grafik forecast
* Evaluasi RMSE

Library

* Streamlit
* TensorFlow
* YFinance
* Pandas
* NumPy
* Scikit-Learn
* Plotly

Menjalankan Aplikasi

Install library:

pip install -r requirements.txt

Jalankan aplikasi:

python -m streamlit run app.py
